| 核心知识点 | 考察能力 | 典型题目 |
| --- | --- | --- |
| **条件类型 (****`extends`****)** | 类型判断、分支逻辑 | `Exclude`、`Flatten` |
| **推断 (****`infer`****)** | 模式匹配、类型提取 | `Shift`、`ReplaceAll`、`FormatDate` |
| **映射类型 (****`[P in K]`****)** | 遍历属性、对象结构重塑 | `Omit`、`Pick`、`Chainable` |
| **模板字面量类型** | 字符串操作、模式匹配 | `Trim`、`ReplaceAll`、`FormatDate` |
| **递归** | 循环操作、处理嵌套结构 | `Trim`、`ReplaceAll`、`Flatten`、`Permutation` |
| **联合类型分配性** | 联合类型处理、全排列 | `Exclude`、`Permutation` |
| **元组 (****`[...T]`****)** | 数组操作、长度计算 | `LengthOfString`、`Shift`、`Pop` |
